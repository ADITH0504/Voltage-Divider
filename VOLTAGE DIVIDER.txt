VOLTAGE DIVIDER

IN THIS NOTE I AM GONNA TALK ABOUT A VOLTAGE DIVIDER WHICH CAN BE HELPFUL FOR OUR FUTURE PROJECTS:

SOME OF THE PURPOSE OF VOLTAGE DIVIDER:
 WE CAN USE A SINGLE POWER SOURCE @ MULTIPLE VOLTAGES FOR LOW LOAD PURPOSES: EG: FROM 12 WE CAN GET 3V, 5V, 9V.
                                         ALSO WE ARE ABLE TO USE THE VOLTAGE DIVIDER EQUATION IN OUR PROGRAM TO DETERMINE OUR BATTERY LEVEL, AN EXAMPLE:
                                              IF OUR BATTERY SUPPLY IS 12v WHICH OUR 3.3V RASBERRY PI CANNOT HANDLE SO WE DIVIDE THAT VOLTAGE INTO 3.3V BY MAKING USE OF THAT DIVIDER EQUATION AND WITH THE HELP OF RESISTOR,
                                                SO ACTUAL VOLTAGE IS 12v AND THAT VOLTAGE IS CORRESPONDED TO 3.3V IN PI, SO AS THE VOLTAGE DROPS FROM 12-0V THE PI GETS CORRESPONDING VOLTAGE BY MAKING USE OF THE DIVIER EQUATION SO THAT WE
                                                   KNOW OUR BATTERY IS AT LOW LEVEL.

                                                  
THIS CAN BE HELPFUL IS MANY PROJECTS INCLUDING IOT PROJECTS.

IMPORTANT THING TO REMEMBER IS, VOLTAGE DIVIDER CAN ONLY BE USED FOR LOW CURRENT APPLICATION AROUND 10 milliampere, SO BASICALLY IT CANNOT BE USED FOR MOTOR, OR HIGH CURRENT LEDS'S.          

ALSO SOME EXTRA TIP: 
WE CAN USE BIDIRECTIONAL LOGIC CONVERTER, WHICH WILL STEP DOWN 5V LOGIC FROM ARDUINO OR 5V SIGNAL TO 3.3V WHICH PI CAN USE.
